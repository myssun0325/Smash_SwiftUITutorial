## 진행방식
---
- 평일 월~금 (2챕터) + 주말(2챕터) 씩 진행해서 2주 예상
- 결과물: 튜토리얼 완성물 + 튜토리얼을 진행하면서 챕터에서 배운 내용 개인의 생각, 챕터에 대한 피드백, 참고 링크, advanced, 더 공부할만한 것, 키워드 등을 공유
- 온라인 모임 (평일챕터 / 주말챕터)
- 눈치보지 않기 (나의 속도 유지하기)
